Diagnóstico de Conocimientos en Precálculo
Este proyecto es una herramienta web interactiva diseñada para realizar un diagnóstico de los conocimientos de precálculo en estudiantes de primer semestre de Ingeniería Industrial.

Descripción del Proyecto
La herramienta busca dar respuesta a la necesidad educativa generada por la falta de estandarización en los conocimientos matemáticos de los estudiantes que ingresan a la universidad. A través de un test interactivo, el estudiante puede identificar sus fortalezas y áreas de mejora, obteniendo un puntaje y un listado de temas a reforzar.

Este proyecto fue desarrollado como parte del Trabajo Final del Máster Universitario en Tecnología Educativa y Competencias Digitales, utilizando un enfoque de desarrollo conceptual asistido por Inteligencia Artificial.

Tecnologías Utilizadas
HTML5: Para la estructura semántica del contenido.

CSS3: Para el diseño visual y la responsividad.

JavaScript (ES6+): Para toda la lógica de la aplicación, interactividad y manipulación del DOM.

Cómo Usar
Para ejecutar este proyecto de forma local, sigue estos pasos:

Clona o descarga este repositorio en tu máquina.

Asegúrate de que los tres archivos (index.html, style.css, y script.js) estén en la misma carpeta.

Abre el archivo index.html en cualquier navegador web moderno (Chrome, Firefox, Edge, etc.).

También puedes acceder a la versión desplegada directamente en el siguiente enlace:
https://jwillsalcedo.github.io/Diagnostico_precalculo/

Funcionalidades Principales
Pantalla de Bienvenida: Solicita el nombre del usuario para personalizar la experiencia.

Examen Interactivo: Presenta una selección aleatoria de 15 preguntas de un banco de 100, cubriendo temas clave de precálculo.

Resultados Detallados: Al finalizar, muestra un puntaje porcentual, un resumen de respuestas correctas y un listado de los temas específicos en los que el estudiante debe mejorar.

Repetir Test: Permite al usuario realizar la prueba de nuevo para medir su progreso.
